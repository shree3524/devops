# $${\color{blue}Create \ Pod}$$

## 1.Install minikube 

**create file** 
````
vim pod.yaml
````
**Inside pod.yaml file write the code**

**then run the following commands**
````
kubectl apply -f pod.yaml
````
````
kubectl get pod nginx
````
````
kubectl describe pod nginx
````
